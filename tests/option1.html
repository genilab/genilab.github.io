Synthetic Data Generation for Clinical Trial Simulation

Unreal Data Bot through which non-technical Researchers can interact to request datasets simulating the expected results of real-world clinical dataset. (in collaboration with School of Medicine, FAU)

Project Sensible Forms

Project Sensible Forms applies Generative AI and Hybrid AI to generate, sequence, and interpret form interactions as adaptive conversations, building an AI-powered Form Engine for dynamic research questionnaires.

Project Sensible Games

Project Sensible Games aims to build a Game Interaction Lab to experiment with Hybrid AI that blends rule-based and generative AI in an interactive, game-like environment. (in collaboration with GamePill.com)
